# Seach Shelter アプリ

## 概要
Seach Shelterは、ユーザーが近隣の避難所を検索し、情報を保存できるAndroidアプリケーションです。Googleマップと連携し、ユーザーが現在地や指定した場所から最寄りの避難所を簡単に検索できます。

## 主な機能
- 近隣の避難所を検索
- Googleマップ上で避難所の位置を表示
- 避難所情報の保存
